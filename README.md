# 株計算ドットコム
株で多分使うかもしれない計算アプリ


## Getting Started
インストール
```bash
bun install
npm install
yarn
```
まず、開発サーバーを起動する：
```bash
bun run dev
# or
npm run dev
# or
yarn dev
```
ブラウザで[http://localhost:3000](http://localhost:3000)を開くと結果が表示される。


## Deploy on Vercel
mainブランチにマージすれば自動的に[株計算ドットコム](https://www.kabukeisan.com/)が更新されます
